# **App Name**: NubMail

## Core Features:

- User Authentication: Secure user registration with email verification and login/logout functionality.
- Domain Management: Add, verify, edit, and delete custom domains with DNS setup instructions and verification status display.
- Email Account Management: Create, list, edit, and delete email accounts per verified domain; set storage quotas.
- Compose Email: Rich text editor for composing emails with attachment support.
- Send & Receive Emails: SMTP integration for sending messages; inbox with sorting and filtering; folders for organizing emails.
- Email Search: Search emails by sender, subject, or content; manage read/unread status.
- Admin Dashboard: Responsive dashboard with key metrics (domains, accounts, storage); domain overview with verification; activity logs and monitoring.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke trust and reliability. Background color: Very light blue (#F0F8FF), creating a clean, uncluttered feel. Accent color: Gentle lavender (#BDB2FF) to highlight key actions.
- Body and headline font: 'Inter', a sans-serif font known for readability, providing a clean and modern appearance.
- Use flat, line-based icons for a modern and clean aesthetic.
- Responsive and modern layout using a grid system.
- Subtle animations and transitions to improve user experience.
